Erin Wilson, Jiatao Cheng, and Adam Canady

Project Description

This project is entitled the "Room Finder Extraordinaire". It takes various inputs and returns lists of rooms or a categorical probability of successfully drawing a particular room. We used data from previous Carleton room draws, and added geolocation from coordinates in Google Maps.

Throughout this project, we leveraged our knowledge of HTML5, CSS3, SQL, and JavaScript on the client side as well as Python, PostgreSQL (and its Python connector, psycopg2), and a pre-configured Apache install on the server side.

Goals and Features

  • Goal: to provide Carleton students with an application that will assist them in the room draw process based on previous room draw data
  • Feature: Find all rooms given certain specifications
  • Feature: Find rooms within certain ranges of a draw number, indicating the probability that the number will get that room
  • Feature: Find rooms near a certain location on campus
  • Feature: Find rooms furthest away from another person's probable draw given their draw number
  • We responded to feedback by adding a feature that lists all the available rooms given certain specifications
